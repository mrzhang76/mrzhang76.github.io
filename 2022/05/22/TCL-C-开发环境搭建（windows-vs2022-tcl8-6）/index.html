发表于 2022-05-22 分类于 折腾的乐趣

配置环境总有坑….

配置TCL环境

安装ActiveTcl

下载地址： https://www.activestate.com/products/tcl/

选择典型模式进行安装，一路下一步即可。

配置TCL环境变量

  • 在系统属性-环境变量中，将x:\ActiveTcl\bin与x:\ActiveTcl\include（x为安装磁盘） 配置搭配系统变量的PATH条目中，重启。

  • 在cmd中执行tclsh命令能打开交互式会话即为配置成功。

准备C++/TCL库

下载地址： https://github.com/flightaware/cpptcl

下载到合适位置，等待调用。

创建VS2022 DLL项目

  1. 打开VS2022，点击创建新项目，选择动态链接库（DLL），设置项目文件位置与项目名称，注意不要勾选“将解决方案和项目放在同一目录中”，完成项目创建。

  2. 编辑项目属性，点击项目-xxx属性（xxx为项目名称），对如下设置进行更改。

    在“VC++目录- 包含目录”中添加条目

    • CPPTCL库头文件目录：“x:\cpptcl-master\cpptcl”
    • TCL头文件目录：”x:\ActiveTcl\include”

    在“VC++目录 - 库目录”中添加条目

        + TCL库文件目录："x:\ActiveTcl\lib"

    在“C/C++ - 常规 - 附加包含目录”中添加条目

        + TCL头文件目录："x:\ActiveTcl\include"

    在”链接器 - 常规 - 附加库目录“中添加条目

    • TCL库文件目录：”x:\ActiveTcl\lib”

  3. 在右侧的项目资源管理器中，进行如下修改

    右键资源文件 ，点击添加现有项，添加在TCL库文件目录中文件：

    • Tcl86t.lib
    • Tclstub86.lib

    右键源文件，点击添加现有向，添加C++/TCL库提供的以下文件：

    • cpptcl.cc
    • cpptcl_runtime.c

  4. 根据实际资源文件位置修改C++/TCL库所提供的文件中的引用信息。
